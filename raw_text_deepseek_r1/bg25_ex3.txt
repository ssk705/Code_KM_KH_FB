public class WelcomeMessage {
    public static void main(String[] args) {
        String name = "Alice";
        System.out.println("Hello, " name "!");  
    }
}
KM:The expression `"Hello, " name "!"` is missing the concatenation operator (`+`) between `"Hello, "` and `name`.
