 public class Test {
    public static void main(String[] args) {
        System.out.println("Start");
        public void display() {  // Bug here
            System.out.println("Hello");
        }
        display();
    }
}
KM:Declaring a method (`display()`) inside another method (`main()`) is invalid in Java (nested methods not allowed).  
KH:Move `display()` outside `main()` (to class scope). Remove `public` if unnecessary.
