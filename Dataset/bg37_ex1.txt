 public class Calculator {
    public static void main(String[] args) {
        int a = 5;
        int b = 10;
    }
    public static void printSum() {
        System.out.println(a + b);
    }
}
KM:Variables `a` and `b` are declared in `main()` and not accessible in `printSum()` (scope error).  
KH:Move `a` and `b` to class-level scope (as fields) or pass them as parameters to `printSum()`.
