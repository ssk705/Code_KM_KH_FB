class Parent {
    void greet() {
        System.out.println("Parent says hello");
    }
}

class Child extends Parent {
    void greet(String name) {
        System.out.println("Child says hello to " + name);
    }
}

public class Main {
    public static void main(String[] args) {
        Parent p = new Child();
        p.greet(); 
    }
}
KM:`Child.greet(String)` is overloaded, not overridden. The superclass method `greet()` remains uncalled due to no parameterless override in `Child`.
KH: Add a parameterless `greet()` method in `Child` that overrides the parent method.
