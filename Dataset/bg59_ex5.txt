[CODE]  
public class InvalidModifier {  
    public static void main(String[] args) {  
        public static int x = 10;  
        System.out.println(x);  
    }  
}  
[OUTPUT]  
KM: Both 'public' and 'static' modifiers are invalid for the local variable 'x'. Local variables cannot have access or static modifiers.  
KH: Remove both 'public' and 'static' modifiers from the declaration of 'x' to make it a valid local variable.
